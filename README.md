ookami-web
==========

Install
------
`git clone https://github.com/ookamiinc/ookami-web.git`  
`cd ookami-web`  
`npm install`  
`bower install`  

Deploy
------
1. Build code `gulp compass`
2. Push to github

| Branch     | Url                     |
|:-----------|------------------------:|
| staging    | staging.playerapp.tokyo |
| master     | playerapp.tokyo         |
